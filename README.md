# 🐶 TinDog

TinDog is a fun, responsive landing page inspired by **Tinder — but for dogs!**  
It’s built with **HTML**, **CSS**, and **Bootstrap 5**, showcasing clean design, responsive layouts, and component-based structure.

---



## 💡 Project Overview

TinDog is a playful mock landing page designed for a fictional app that helps dogs find their perfect companions.  
This project demonstrates:
- Responsive design using **Bootstrap 5**
- Use of **grid system**, **flexbox**, and **utility classes**
- Implementation of **cards**, **buttons**, and **gradient backgrounds**
- A clean, modern UI with structured sections

---

## ✨ Features

- 🎨 Animated gradient hero section  
- 🐕 Catchy slogan and mobile mockup display  
- 💼 Feature cards highlighting app benefits  
- 💬 Testimonial section with user feedback  
- 💰 Pricing plans (Chihuahua, Labrador, Mastiff)  
- 📱 Fully responsive on all devices  

---

## 🖼️ Sections Overview

| Section | Description |
|----------|--------------|
| **Title Section** | Main heading, CTA buttons, and mockup image |
| **Features** | Highlights the app’s simplicity and advantages |
| **Testimonial** | Displays user feedback with press logos |
| **Pricing** | Three-tier pricing cards for dog owners |
| **Footer** | Clean footer with navigation links |
